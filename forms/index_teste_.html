Foram criadas duas intenções de substituição no nível 1 com o Machine Learning ativado. Uma para lidar com palavras que não representam ações que estão contidas nas opções do menu das perguntas frequentes no módulo de usuário. Para isso a intenção não possui nenhuma frase de treinamento, base, pois assim quando o chat não conseguir corresponder a nenhuma intenção, o chat irá corresponder a ela. Já a outra para lidar com as opções do FAQ do menu secundário, para que o usuário não tenha acesso às opções do menu secundário, enquanto estiver no nível 1. Essa intenção possui como frase de treinamento os numeros de 1 a 5, que representam no menor ao maior numero utilizado nas como frase de treinamento de alguma intenção de menu secundario
